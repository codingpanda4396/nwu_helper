# 运营 SOP

## 每日流程

1. 管理员检查待审核帖子，通过或拒绝社区投稿。
2. 管理员检查反馈列表，处理用户建议和投诉。
3. 管理员检查商家状态，审核新商家、调整排序。
4. 管理员查看数据概览，关注用户增长和活跃度趋势。

## 周复盘

- 看商家列表，优化封面图、简介和排序权重。
- 看社区帖子互动，识别热门话题和活跃用户。
- 看用户反馈，收集产品改进建议。
- 看数据分析页面，关注用户增长、留存和转化漏斗。
- 更新轮播图和活动，保持首页内容新鲜。

## 内容管理

- **轮播图**：定期更新首页 Banner，配合活动或推广。
- **活动**：为商家创建限时活动，设置状态和时间窗口。
- **社区帖子**：审核学生投稿，维护社区氛围。
- **微信入口**：确保微信二维码和引导文案有效。
