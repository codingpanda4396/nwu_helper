# 功能边界报告

## 架构概述

系统采用 monorepo 架构，包含以下三个独立应用：

| 应用 | 技术栈 | 端口 | 说明 |
|------|--------|------|------|
| API | Fastify + Prisma | 4000 | 后端服务 |
| Admin | React + Vite | 5175 | 管理后台 |
| Student | UniApp + Vue3 | 5174 | 学生端（H5 & 小程序） |

## 功能边界

### 学生端（UniApp）

- 浏览类目、商家列表和商家详情
- 查看轮播图、活动推荐
- 美食、服务、社区、个人中心
- 支持 H5 和微信小程序

### 管理后台（React）

- 类目管理：增删改查本地生活类目
- 轮播图管理：首页 Banner 配置
- 服务分类管理：生活服务子分类
- 社区管理：帖子审核、状态管理
- 微信入口配置：首页微信引导
- 数据看板：商家数量、活动数量、待审核帖子统计

### 后端 API

- 公开接口（无需认证）：
  - `/api/public/*` - 学生端使用的公开数据
- 管理接口（需认证）：
  - `/api/admin/*` - 管理后台使用的接口

## 已删除功能

以下功能已从管理后台中移除：

- 商家管理：商家列表、商家表单
- 活动管理：活动列表、活动表单
- 美食商家管理：美食商家列表
- 服务商家管理：服务商家列表

## 未覆盖范围

- 完整商家后台，包括商家登录、经营数据、资料维护、优惠券维护、核销操作
- 在线支付、微信支付、自动分账、复杂交易结算和财务对账
- 微信登录、手机号授权、实名学生认证和用户账号体系
- 原生客户端发布流程、平台审核材料和生产 AppID 配置
- 复杂内容审核、风控、反作弊、营销归因报表和多角色权限矩阵
